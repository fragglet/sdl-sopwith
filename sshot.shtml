I recently bought a Psion Revo Palmtop, and decided to port sopwith to it. The screen is a bit squished as it only has 160 lines (and sopwith is designed to run on a 320x200 screen). I have expanded the field of view to fill the entire screen. I am in the process of building a linux distribution for the Revo. You can find more about this here.

The Psilinux port is now almost complete. I wrote the keyboard code and improved the graphic code a bit.

Sopwith running under linux in the Fluxbox and Metacity window managers.
